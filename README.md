# Random
El proyecto es un `Fork` del repositorio [Random](https://github.com/jquemada/random/) del profesor Juan Quemada.

Random es un super programa `open source` hecho como un ejemplo que funciona sobre node, que genera un número aleatorio entre 0 y 1, y lo muestra por la consola.

## Instalar
Ejemplo sencillo para copiar Random:

* **Usar GitHub**: `usar la interfaz de GitHub para clonar el repositorio`.
* **Descargar el fichero zip**: [releases](https://github.com/taferk/random/releases). 
* **Descargar el repositorio desde git**: `git clone git://github.com/taferk/random`.
